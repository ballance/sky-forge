# Neighborhood Drone Mapping System

## Overview
Automated drone mapping system for a 96-house neighborhood using the Potensic Atom 2 drone.

## Potensic Atom 2 Specifications
- **Camera**: 4K (3840x2160) video, 12MP photos
- **Flight Time**: ~32 minutes per battery
- **Max Speed**: 16 m/s
- **Max Altitude**: 120m (400ft) - FAA limit
- **GPS**: Yes, with Return-to-Home
- **Gimbal**: 3-axis stabilized
- **Storage**: MicroSD card support
- **Control Range**: Up to 6km

## Key Features for Mapping
- GPS positioning for waypoint navigation
- Stable hover capability
- 4K camera suitable for photogrammetry
- Good battery life for extended missions
- Compact size for residential area operations

## Project Components
1. **Flight Planning**: Automated grid pattern generation
2. **Image Capture**: Optimized settings for mapping
3. **Processing Pipeline**: Orthomosaic and 3D model generation
4. **Quality Control**: Coverage verification and data validation

## Safety and Legal Considerations
- Check local airspace restrictions
- Obtain necessary permissions/waivers
- Notify neighbors about mapping activities
- Follow FAA Part 107 regulations (if in USA)
- Maintain visual line of sight
- Avoid flying over people

## Recommended Mapping Parameters
- **Altitude**: 60-80m (200-260ft) for residential detail
- **Overlap**: 70% front, 60% side
- **Speed**: 5-8 m/s for sharp images
- **Time of Day**: Mid-morning or late afternoon (avoid harsh shadows)
- **Weather**: Overcast days ideal (even lighting)
